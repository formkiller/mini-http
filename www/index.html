🚀 MiniHTTP Server is Running!

What is this?

A lightweight HTTP/1.1 server built from scratch using the muduo network library (C++11).

Features

  • Custom HTTP Parser — State machine that processes requests byte by byte from muduo::Buffer
  • URL Routing — map-based route table: /, /health, /about
  • Static Files — Serve HTML/CSS/JS/images with proper MIME types and 404 handling
  • Keep-Alive — HTTP/1.1 persistent connections with automatic timeout
  • Multi-threaded — Uses EventLoopThreadPool (one loop per thread)
  • Security — Directory traversal protection (/../ returns 403)

Endpoints

MethodPathDescription
GET/This page (index.html)
GET/aboutAbout the server
GET/healthHealth check (JSON)
GET/style.cssCSS file (MIME: text/css)
GET/nonexistent404 Not Found demo

How to test

# Start the server
./bin/http-server 8080 4 ./www

# Test with curl
curl -v http://localhost:8080/
curl http://localhost:8080/health
curl http://localhost:8080/about
curl -I http://localhost:8080/          # HEAD request
curl http://localhost:8080/nonexistent  # 404

# Stress test with ApacheBench
ab -n 1000 -c 10 http://localhost:8080/

# Test security
curl http://localhost:8080/../etc/passwd  # Should return 403
